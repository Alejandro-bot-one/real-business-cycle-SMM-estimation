# Real Business Cycle Model: SMM Estimation and Simulation

This repository contains the code and data to estimate, calibrate, and simulate a standard Real Business Cycle (RBC) model. The primary contribution of this project is the empirical estimation of the technology shock parameters (the Solow Residual's autoregressive persistence and standard deviation) using the Simulated Method of Moments (SMM) to match US macroeconomic data.

## Project Overview

The Real Business Cycle (RBC) theory posits that business cycle fluctuations are efficient responses to real exogenous shocks, primarily total factor productivity (technology). 

Instead of relying solely on standard steady-state calibration or direct OLS on the Solow residual, this project utilizes the Simulated Method of Moments (SMM). By minimizing the distance between the empirical moments of the US economy (e.g., variance and autocorrelation of the GDP cycle) and the theoretical moments generated by the model, we accurately estimate the shock processes driving the business cycle.

## Repository Structure

The workflow is divided into data acquisition, model simulation, and parameter estimation. 

* `us_data.csv`: The processed dataset containing the cyclical components of US macroeconomic variables used as empirical targets for the SMM procedure. Although controversial, I have used the HP Filter to extract the business cycle time series.
* `RBC_smm.mod`: The Dynare model file structured specifically for the SMM estimation. It contains the log-linearized equations of the RBC model and computes the theoretical moments.
* `SMM.m`: The main MATLAB script that executes the Simulated Method of Moments. It iteratively calls Dynare, compares the simulated moments to the empirical data in `us_data.csv`, and optimizes the parameters.
* `simulations.mod`: A standalone Dynare file used to generate Impulse Response Functions (IRFs) and simulate the economy's reaction to supply (technology) and demand (government) shocks using the optimized parameters.

## Methodology

### The Model
The framework is a stochastic neoclassical growth model featuring:
1. Endogenous labor supply and capital accumulation.
2. A Cobb-Douglas production function subjected to an AR(1) technology shock.
3. Fully flexible prices and rational expectations.

### Calibration and SMM
Standard structural parameters (discount factor, capital share, depreciation rate) are calibrated using long-term steady-state ratios. 

The parameters of the exogenous shock processes (persistence and volatility) are estimated via SMM. The algorithm searches for the parameter vector that minimizes a weighted distance between the empirical variance-covariance matrix of the data and the simulated variance-covariance matrix generated by Dynare.

### Evaluating Model Performance
The project also evaluates the standard RBC model's successes and failures, specifically addressing output/investment volatility and the "Labor Puzzle" (Paul Klein's critique regarding the empirical correlation between real wages and hours worked).

## Prerequisites and Installation

To run this project, you will need the following software:
* **Python 3.x**: With `pandas` and `numpy` installed (for `get_data.py`).
* **MATLAB** (or Octave).
* **Dynare**: A pre-processor and collection of MATLAB routines for solving DSGE models. Ensure Dynare is properly added to your MATLAB path.
